Начало

Для создания светового меча, первое, что вам нужно будет сделать - это собрать электрическую цепь из его компонентов.

Электри́ческая цепь — совокупность устройств, элементов, предназначенных для протекания и потребления электричества.

Обычно её составляющие это - источник тока, провода и потребители электрического тока.

В вашей цепи источником тока будет аккумулятор, он будет хранить заряд энергии для работы остальных компонентов.

Потребителем будет излучатель - он преобразует получаемый заряд в лазер.

Проводником энергии являются провода - нужно соединить плюс с минусом, минус с плюсом.
